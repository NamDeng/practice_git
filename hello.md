# hello git

## git 명령어 요약
- clone : 원격 저장소 복사
- add : 작업 스테이지 영역에 작업 파일 추가
- commit : 세이브, 스테이지 영역의 파일을 가지고 커밋을 만들 수 있다.
- push : 원격 저장소에 커밋을 업로드한다.
